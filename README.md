Glinde - OpenGL game engine experiment
==================================

This is an experimental project launched to study how a 3D engine can be built

## Requirements

 * blender
 * g++
 * glfw-3
 * glew
 * libpng
 * libzip
 * libjansson
 * wand (built tool)

## Compile

    make

## Run

    __targets_-xxx/glinde
